# Karl Solution Scaffold

This is a scaffolded .NET 10.0 solution for Karl, including:

- Karl.Abstractions
- Karl
- Karl.Transport.MailKit
- Karl.Transport.File
- Karl.Template.Scriban
- Karl.Extensions.Microsoft
- Karl.Cli

You can open this in Visual Studio or with the `dotnet` CLI.

```bash
dotnet build
```
